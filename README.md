## Getting Started

To get a local copy up and running follow these simple example steps.

- Clone this repository with git clone`git@github.com:Berkay-akbas/hello-rails-react` using your terminal or command line.
- Change to the project directory by entering : <br>
  `cd hello-rails-react` in the terminal.

### Setup

You can clone this repository or simply download the files as a .zip
If you want to set up a personal repository based on this one, you can as well fork it.

### Run test for linter:

```
rubocop -A
```

### Start the app:

```
ruby main.rb
```

### Install RSpec

```
gem install rspec
```

### Run tests:

```
rspec .
```
